# SteelBullets

<h3>TPP shooter prototype based on UE4, C++ and UE shooter assets pack</h3>

Features:
- Components
- Two weapons with different behaviour
- Falling damage
- Autoheal
- UI
- Sounds
- Animations
- VFX
- Basic AI
- Pickups
